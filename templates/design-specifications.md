# Design Specifications

## HDPM Instagram Carousel Design System

### Brand Identity
- **Logo:** HDPM windswept juniper tree
- **Primary Color:** Forest Green #2C5530
- **Secondary Color:** White #FFFFFF
- **Accent:** Light gray for backgrounds

### Typography
- **Headings:** Bold, sans-serif, forest green
- **Body:** Regular, sans-serif, black or dark gray
- **CTAs:** Bold, white text on forest green buttons

### Layout Grid
- **Format:** 9:16 vertical (Instagram Story/Carousel)
- **Dimensions:** 1080x1920px recommended
- **Safe Area:** 60px margins from edges

### Slide Types

#### 1. Hero Slide (Slide 1)
- Large property image background
- Property type headline (top)
- Price (prominent, center-top)
- Address (bottom)
- HDPM logo (corner)

#### 2. Interior Slides (Slides 2-5)
- Full-bleed room photo
- Descriptive headline overlay (top)
- HDPM logo (bottom corner)
- Minimal text, focus on imagery

#### 3. Features Grid (Slide 6)
- White or light background
- Icon + text combinations
- 6-8 key features
- HDPM logo at top

#### 4. Additional Details (Slides 7-8)
- Supplementary property photos
- Context-specific headlines
- Supporting feature callouts

#### 5. Agent Slide (Slide 9)
- Professional agent photo
- Name and title
- Contact information (phone)
- "Schedule a tour" CTA
- HDPM branding

#### 6. CTA Slide (Slide 10)
- Bold headline
- Primary CTA button (phone/text)
- Secondary CTA (online application)
- HDPM branding
- Availability status

### Two-Tier Positioning

#### Budget Tier
- **Aesthetic:** Warm, friendly, accessible
- **Imagery:** Clean, well-maintained, practical
- **Copy Tone:** Straightforward, value-focused
- **Features Focus:** Included utilities, practical amenities

#### Premium Tier
- **Aesthetic:** Modern, sophisticated, aspirational
- **Imagery:** High-end finishes, luxury details
- **Copy Tone:** Elevated, lifestyle-focused
- **Features Focus:** Premium amenities, location benefits

### Color Usage by Tier

**Budget:**
- Forest green primary
- Warmer accent tones
- Focus on clarity and readability

**Premium:**
- Forest green primary with gradient accents
- Cooler, more sophisticated palette
- Emphasis on elegance and space

### Image Requirements
- **Resolution:** Minimum 2K (2048x pixels)
- **Format:** PNG for graphics, JPEG for photos
- **Quality:** High compression acceptable for web
- **Optimization:** <1MB per slide ideal

### Accessibility
- Minimum contrast ratio 4.5:1 for text
- Font size minimum 24px for body text
- Headings 36px+ for readability on mobile
- Alternative text for screen readers (when applicable)

---

**This design system ensures consistent, on-brand carousel production across all HDPM properties.**
